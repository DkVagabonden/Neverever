# Neverever
This is a small "Never Have I Ever" game-project. 
Live version at http://neverever.96.lt/


Features
=======
PHP Login<br>
Admin panel with user interface<br>
Create new questions<br>
Create new users<br>
Overview of how many questions made and shown<br>
